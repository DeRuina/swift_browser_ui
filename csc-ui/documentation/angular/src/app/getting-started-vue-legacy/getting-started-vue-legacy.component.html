Usage in Vue 2.x

1. Initialize a new Vue project

These commands will first install the Vue Cli globally, and then use it to initialize a new project. Select Vue 2 preset when prompted.

2. Install the required dependencies

This command will install the CSC Design system component library csc-ui and the directive @cscfi/csc-ui-vue2 which allows the components to support 2-way model binding.

3. Configure the project to utilize the components by adding these lines to main.js

Now you should be able to use the CSC Design system components in your project. Please note that when using the @cscfi/csc-ui-vue2 with Vue 2.x you need to bind your models with v-csc-model instead of v-model .
